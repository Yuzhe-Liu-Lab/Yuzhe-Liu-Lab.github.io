## Introduction

Brain-RBF provides a workflow for morphing a baseline finite element head model to match target brain morphology using radial basis function thin-plate spline (RBF-TPS) interpolation. The repository is designed around node-coordinate input from the THUMS head model, allowing users to extract nodal coordinates from the model keyword file and generate morphology-adapted brain meshes for downstream finite element analysis.

## 中文介绍

Brain-RBF 提供了一套基于径向基函数薄板样条（RBF-TPS）的有限元头部模型网格变形流程，可将基准模型变形到目标脑形态。该仓库以 THUMS 头部模型的节点坐标为主要输入，用户可从模型 K 文件中的节点信息提取坐标，并生成与个体脑形态相匹配的模型，用于后续有限元仿真与脑应变分析。
